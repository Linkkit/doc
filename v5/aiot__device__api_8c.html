设备注册感兴趣的topic，接收到该topic的消息，执行对应回调

参数
[in]device设备句柄
[in]topic需要注册的topic
[in]callback回调函数
[in]sub是否需要向云端发送订阅消息，设备只需要订阅一次即可，重启后一样有效 [1, 向云端发送订阅消息] [0, 不向云端发送订阅消息]
[in]userdata执行回调时返回给用户
返回
int32_t
返回值
<STATE_SUCCESS执行失败, 更多信息请参考aiot_state_api.h
>=STATE_SUCCESS执行成功
