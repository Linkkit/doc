枚举类型说明

enum aiot_device_status_type_t

device状态变化类型

枚举值
AIOT_DEVICE_STATUS_CONNECT 

当实例第一次连接网络成功时, 触发此事件

AIOT_DEVICE_STATUS_RECONNECT 

当实例断开网络连接后重连成功时, 触发此事件

AIOT_DEVICE_STATUS_DISCONNECT 

当实例断开网络连接时, 触发此事件

enum aiot_device_msg_result_type_t
枚举值
AIOT_DEVICE_SEND_MESSAGE_RESULT 

消息发送的结果

AIOT_DEVICE_REGISTER_TOPIC_RESULT 

订阅感兴趣的消息的结果

AIOT_DEVICE_UNREGISTER_TOPIC_RESULT 

取消订阅消息的结果

设备注册感兴趣的topic，接收到该topic的消息，执行对应回调

参数
[in]device设备句柄
[in]topic需要注册的topic
[in]callback回调函数
[in]sub是否需要向云端发送订阅消息，设备只需要订阅一次即可，重启后一样有效 [1, 向云端发送订阅消息] [0, 不向云端发送订阅消息]
[in]userdata执行回调时返回给用户
返回
int32_t
返回值
<STATE_SUCCESS执行失败, 更多信息请参考aiot_state_api.h
>=STATE_SUCCESS执行成功
